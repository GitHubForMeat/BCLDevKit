APPLICATION STATUS

Streaming Mode
Shows the current streaming mode of the application, and may take the following values:

- Send Always
In this mode the application is always sending a stream to the configured destination IP and port.

- Send on Level
In this mode the device will stream if the input audio level goes above the configured threshold level.

- Send on I0/DI0
The device will stream if will stream if the I0 (Annuncicom100/Legacy) or DI1(Annuncicom1000/Exstreamer1000) is activated.

- Respond
The device will stream back as long as there is a stream being received.


Remote Party
Shows the IP and port of the current remote peer.


AUDIO STATUS
Shows the status of the following audio parameters:
- Volume:
Displays the current volume in %
- Input Peak Level
Displays the current input peak level
- Output Peak Level
Displays the current output peak level
